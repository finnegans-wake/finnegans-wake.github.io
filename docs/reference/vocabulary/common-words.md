# Common Wake Words

Certain words and phrases recur throughout *Finnegans Wake*. Learning these common terms helps you navigate the text.

## Character Names

### HCE Names

The father appears under countless names with H-C-E initials:

| Name | Page(s) | Notes |
|------|---------|-------|
| **Humphrey Chimpden Earwicker** | Throughout | Full formal name |
| **Here Comes Everybody** | 32 | His universal identity |
| **Haveth Childers Everywhere** | 535 | His boast |
| **Howth Castle and Environs** | 3 | Opening; landscape as body |
| **Haroun Childeric Eggeberth** | 4 | Early variant |
| **Mr. Porter** | III | His "realistic" name |
| **Persse O'Reilly** | 44–47 | In the ballad (earwig/perce-oreille) |
| **Festy King** | I.4 | At his trial |
| **Finnegan** | 3–29 | As the fallen builder |

### ALP Names

The mother/river appears as:

| Name | Page(s) | Notes |
|------|---------|-------|
| **Anna Livia Plurabelle** | Throughout | Full formal name |
| **Annah the Allmaziful** | 104 | Invocation |
| **Mrs. Porter** | III | Her "realistic" name |
| **The Liffey** | I.8 | As river explicitly |
| **Biddy** | 10–11 | As the hen |
| **Biddy Doran** | 111 | The hen's full name |

## Recurring Phrases

### Opening/Closing

| Phrase | Page | Meaning |
|--------|------|---------|
| **"riverrun"** | 3, 628 | The flow begins/continues |
| **"past Eve and Adam's"** | 3 | Adam and Eve Church, Dublin; Eden |
| **"Sir Tristram"** | 3 | Tristan legend begins |
| **"A way a lone a last a loved a long the"** | 628 | The dissolving ending |

### Thunder/Fall

| Phrase | Meaning |
|--------|---------|
| **"bababadalgharaghtakammina..."** | Thunder; the fall |
| **"The fall"** | Original sin; Finnegan's tumble |
| **"Foenix culprit"** | Felix culpa; happy fault |
| **"Humpty Dumpty"** | The fallen egg-man |

### Time/Cycle

| Phrase | Meaning |
|--------|---------|
| **"The Vico road"** | The cyclical path (also a Dublin road) |
| **"In the beginning"** | Genesis; every beginning |
| **"Teems of times"** | Teams/themes of times; recurrence |
| **"Finn, again!"** | Resurrection; the cycle |

## Sound Words

### Thunder Sounds
- **Bababadal...** — Crash, rumble
- **Rrrrrr** — Rolling thunder
- **Crack** — Lightning

### Water Sounds
- **Liffey** — Flowing
- **Hitherandthithering** — Water movement
- **Splash** — Impact

### Animal Sounds
- **Quark** — Gull cry
- **Cock-a-doodle** — Rooster (dawn)
- **Moo** — Cow

## The Sigla in Text

Joyce used symbols (sigla) which sometimes appear as:

| Symbol | Meaning | Text Form |
|--------|---------|-----------|
| ⌂ | HCE | House shapes, "M" |
| Δ | ALP | Triangle, delta |
| ⊏ | Shem | Open bracket |
| ⊐ | Shaun | Reversed bracket |
| ⋀ | Issy | Caret, divided |
| X | The Four | Cross |

## Phrases to Recognize

### Greetings

| Phrase | Meaning |
|--------|---------|
| **"Howth ye?"** | How are you? + Howth |
| **"Good morrow"** | Good morning |
| **"Nighty night"** | Goodnight |

### Exclamations

| Phrase | Meaning |
|--------|---------|
| **"O!"** | Exclamation; also zero (0) |
| **"Hohoho"** | Laughter; HCE |
| **"Hahahaha"** | Laughter |
| **"Alas!"** | Sorrow; also ALP |

### Questions

| Phrase | Meaning |
|--------|---------|
| **"What?"** | Often "Wha?" or "Hwat?" |
| **"Where?"** | "Ware?" |
| **"Who?"** | "Hoo?" (also owl sound) |
